## 🛠️ 技术架构总结

### 后端架构
```
Node.js + Express.js + TypeScript
├── API层: RESTful API设计
├── 服务层: 业务逻辑封装
├── 数据层: Sequelize ORM
└── 监控层: 系统监控和日志
```

### 前端架构
```
React + TypeScript + Ant Design
├── 组件层: 可复用UI组件
├── 页面层: 业务页面实现
├── 状态层: React Hooks状态管理
└── 工具层: 工具函数和配置
```

### 数据库架构
```
Microsoft SQL Server
├── hotdog2030: 统一数据平台
├── cyrg2025: 源数据1
├── cyrgweixin: 源数据2
└── ETL: 数据同步和转换
```
