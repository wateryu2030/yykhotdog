## 📊 技术架构

### 后端技术栈
- **框架**: Node.js + Express.js
- **数据库**: Microsoft SQL Server
- **ORM**: Sequelize
- **语言**: TypeScript
- **API文档**: Swagger
- **日志**: Winston

### 前端技术栈
- **框架**: React + TypeScript
- **UI库**: Ant Design
- **图表**: Chart.js + React-ChartJS-2
- **路由**: React Router
- **状态管理**: React Hooks

### 数据库架构
- **主数据库**: `hotdog2030` (统一数据平台)
- **源数据库**: `cyrg2025`, `cyrgweixin`
- **数据同步**: ETL流程 + 实时同步
